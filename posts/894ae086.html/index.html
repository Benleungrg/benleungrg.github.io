Blender Project UV Learning Log

2023-04-09 logs

Basic Workflow:

  1. Add new material.
  2. Open Shader Editor in Editor Type at the top left.
  3. Drag the reference image into Shader Editor and link it to the Base Color of the geometry.
  4. Open two windows, one for modeling and another for UV Editor.
  5. In edit mode, select the face of the geometry, press the U key, and click “Project from View”.

Make Reflection For Glass etc:

  1. add a new material slot.
  2. click and assign the new slot to the original material reference. (material ball drawer icon)
  3. In object mode, click the fake user button (the shield icon) to unlink it from the original one, which will create a copy.
  4. Increase the Specular and decrease the Metallic and Roughness to make the surface glossy.
  5. Select and assign the material to the glass part.
  6. Add surface imperfection.

Reference Image Blending

  • If the resolution is too low, upscale it to 2k, paint over the edge to make it sharp, or use a blending brush to repaint a little.
  • Gather more references for detail like logo or branding, photoshop them together.

Other Notes

  • If something didn’t function properly, try switching to Edit Mode/Object Mode.
