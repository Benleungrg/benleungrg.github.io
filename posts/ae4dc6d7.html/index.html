【HAM】FT8CN + 安卓手机 + 拓展坞解决 FT891 无法发射

2025-06-25 Ham

经过前文：【Ham】八重洲 FT-891 玩 FT8 模式的记录和踩坑。

已经成功验证我自制的音频线配合笔记本电脑可以正常工作，由于电脑得另作他用所以决定试下手机端的 FT8CN app。

根据网上的教程和信息，只需要用一台安卓手机，然后插入 type-c 口转 3.5mm 音频口，再插入连接电台的音频线，电台调整到 VOX 声控模式，手机播放 FT8 音频，传输到电台启动发射即可，然而实际操作还是踩坑。

为什么手机要插 TypeC 转 3.5mm 音频口？

基本上看到的视频都使用转接线，可我手机自带就有 3.5mm 的耳机孔啊，干嘛要转接，遂试，发现开启电台 VOX，VOX 的类型设置成 DATA， FT8CN 也设置控制模式为 VOX，可输出音频的时候无法启动音频的发射。

捣鼓了各种设置都无法测试成功。

试验下来仅能确认：

  • 音频线是正常的，它插入手机后，把电台扬声器关掉，电台还是能解码信号，发射时，手机扬声器没有声音，意味着声音正在通过音频线传输到电台，但是不知为何没能通过 VOX 将电台的发射给启动。

于是怀疑：难道就是要 TypeC 线转接音频，利用手机的 OTG 功能才能配合 APP 传输音频？

然而 PDD 斥 3 元巨资买了一根回来测试，我的手机压根不能通过 C 口转 3.5 传输音频，接收和输出都不行。

这个问题到现在都没有头绪。

USB 拓展坞登场

家里有一个 PDD 买闲置的 USB 拓展坞，可惜当时没有买带 3.5mm 孔的那款，我这个是 USB 口加 Type-C 的，但是也凑合用。

手机→拓展坞→ CAT 控制线→电台

发现无法在 FT8CN 控制电台切换频率，但是配合 ACC 音频线倒是可以发射了。

在 FT8CN 中的控制模式需要改成 RTS 即可

然后其它设置跟电脑的差不多，选好型号等就行了。

但是音频线还有一个坑

当你用音频线直接插入手机耳机孔时，会发现音频并没有连接电台，收音是通过手机的麦克风，输出是扬声器。

此时回想起来在 B 站看到一个相关视频，UP 主遇到了同样的问题。

他的解决方法是：将耳机插头，半插入耳机孔，观察手机状态栏，当耳麦图标出现后，再完全插入耳机孔。注意一定是要耳麦图标，而不是耳机图标，两者有区别，如果只显示耳机图标，此时虽然可以正常收音，但是发射音频时是没有输出的。
