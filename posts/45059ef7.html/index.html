【Ham】八重洲 FT-891 玩 FT8 模式的记录和踩坑

2025-05-25 Ham

电台设置

选择 DATA USB 模式而不是纯 USB 模式，因为不用每次玩完 FT8 都要手动进入菜单，将 PTT 切换成 Mic ，进入 DATA 模式所有参数都准备好了，会方便点。

  • 05-06 CAT RATE: 9600

  • 05-08 CAT RTS : DISABLE

  • 08-01 DATA MODE: OTHERS

  • 08-03 OTHER DISP : 1500

  • 08-04 OTHER SHIFT : 1500

  • 08-05 DATA LCUT FREQ: OFF

  • 08-07 DATA HCUT FREQ: 4000

  • 08-09 DATA IN SELECT: REAR

  • 08-10 DATA PTT SELECT: RTS

  • 08-12 DATA BFO: USB

  • 16-14 DATA DATA GAIN: 10

将以下增益关闭：

  • AGC

  • NAR

  • IPO

  • NB

开启 WIDTH 并调到：3000 kHz

软件设置

  • 确保安装好 USB 线的驱动，最好将电台断电再拔插控制线。

  • 打开电脑的设备管理器，找到 USB 设备，当 USB 线被成功识别会显示两个串口，要分别记住它们。

  • 打开软件进入设置，选好自己的无线电设备型号

  • CAT 控制板块的串行端口：选择刚在设备管理器中末尾带 Enhanced COM Port 的串口。

  • 波特率：选择与你电台设置的相同，我这里设置为 9600。

  • PTT 方法：RTS； 端口：选择刚在设备管理器中末尾带 Standard COM Port 的串口。

  • 模式：数据 / 封包

设置完成后，以下为踩到的坑：

干扰

  • 自制了一条音频线，又买了条普通带一个干扰磁环的打印机线，想着 RFI 应该不严重，结果是一发射就会掉 COM 口，然后无法 USB 口无法识别控制线，需要重启电台 + 重插线，一开始买了 5 个小尺寸的抗干扰磁环扣，扣了三个在打印机线，两个在音频线上，发射一超过 10W 还是有几率掉 COM 口。于是又买了大尺寸的磁环卡扣，卡了 4 个在打印线上，每个扣都饶了两圈，剩下小尺寸卡扣全部卡音频线上了，功率拉上去，连接基本也没出过问题了。
  • 网上提到将电台接地可以有效消除干扰，但苦于我的居住环境，只能就近引从插座引一条地线接到机器的接地口上，但效果不明显。并且目前我只用电池供电，并且不是固定站台，在磁环成功解决问题后就放弃接地了。

ALC 过高

在学习过程中我了解到，在发射 FT8 的时候，应该关注 ALC 表的值，如果太高将会导致你的信号过调制，ALC 表显示拉满，会成为频率的干扰源。理想的状态是只起一点点表。

看了很多别人的设置和求助帖，他们提到可以降低电脑输出的音量来降低 ALC，或者调整软件的功率条来压低 ALC，但是经过我反复测试，无论是你调整音量或是降低功率条，都会导致输出功率的降低。

比如我设置了 15W 发射时，ALC 破表，当调整音量或功率条将 ALC 压到一个勉强可以接受的程度（大概占整个条的 1/3），此时观察机器的功率输出，会呈现出功率在～2W 到～8W 来回跳动的情况。

后来解决的方法是去机器菜单：16-14 DATA DATA GAIN，将默认值 50 降低到 10。

然后功率输出就稳定了，并且 ALC 表只起了一小格。
