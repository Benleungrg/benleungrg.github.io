【HAM】 FT-891 + 迷你主机实现远程玩 FT8 的方案

2025-07-22 HAM

传播一直不好，只能晚上下班开机，这时段就连 FT8 都没啥信号，于是萌生了远程玩 FT8 的想法。

需要的只有：

  • 迷你主机

  • 智能插座 / 通断器

  • 音频线一分二转接

  • 远程软件

  • 电台控制软件

迷你主机

手头虽然有主力台式机但是考虑这种机器长期挂机会比较费电，而且干正事的时候会互相冲突。

于是感觉最有性价比还是直接淘台小主机，经过一番简单搜寻决定搞台 J1900 CPU 的迷你主机。咸鱼上 该 U 主机，带硬盘内存和电源的总体都在 230 + 往上，但是找到个个人卖家的带原装电源， 8G 内存 + 128G 固态硬盘，只要 188，简单对比了下就下手了，只需要看准有耳机麦克风插口，USB 插口，以及有没有无线网卡就完事了。

性能应该刚好够用，但是如果再挂上 Grid Tracker 应该就有点吃力了。

收到机器测试没有翻车，因为懒得折腾然后也刚好有现成的刷机盘，所以直接灌一个官方精简版 win11，进入 BIOS 在 POWER 里设置 [After power loss] 为 [Power on]，也就是来电开机。

运行 JTDX，远程软件，FLRIG，CPU 负载在 60~80% 左右。

但是后期测试中发现，来电开机功能不能每次都成功，成功几次之后就会开始出现无法进入系统的情况，进入 BIOS 看系统时间被重置，而之前打开的来电开机功能被重置成了来电不开机，判断为 BIOS 电池没电了，买电池换上解决（买纽扣电池记得留意是三线接口还是两线，虽然我没留意买到两线但是只需要按原来的正极（红线）接好也照样能用。）

智能插座 / 通断器

可以网购智能插座，但是因为我有闲置的用于灯具的米家通断器，以及小米的网关，所以直接改造一个智能插排，穷玩方案不建议模仿。要确定通断器的电压和最大承受电流是否能够承受。并且该通断器为蓝牙 mesh 设备，必须有一个蓝牙 mesh 网关才可以接入米家，一些小爱音箱自带蓝牙网关功能。

接线如图，排插必须有地线，否则电台的电源没有地线会导致外壳带电。

如此可以实现：

使用米家开启通断器→排插通电→电台来电开机→迷你主机来电开机并开机自启远程软件。

不使用的时候迷你主机正常关机→米家关闭通断器→排插断电→电台和迷你主机断电。

远程软件

之前用 Todesk ，局域网内用不错，但是一出门通过 P2P 连接就时不时断我连接，说通道拥挤，于是选择向日葵，免费，选择十五帧够用了，也不会踢我下线（目前为止）

电台控制软件

FLRIG，用于观察电台的驻波，功率和 ALC，实时调整电台的功率 RF 之类的，反正该有的都能调整。

在 JTDX 中，电台型号直接选择 FLRIG 可以实现软件联动，否则两个软件只有其中一个软件可以连接到电台的端口。

  1. 迷你主机
  2. 智能插座 / 通断器
  3. 远程软件
  4. 电台控制软件
